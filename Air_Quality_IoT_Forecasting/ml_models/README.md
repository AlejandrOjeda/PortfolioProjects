# Machine Learning Models

This folder contains the notebooks used for the analysis and development of the PM2.5 forecasting models.

Contents:

- estadistica_descriptiva.ipynb  
  Exploratory data analysis of the environmental data.

- regresion_lineal.ipynb  
  Baseline model using linear regression.

- forecasting_randomforest.ipynb  
  Random Forest model used for PM2.5 forecasting.

- forecasting_xgboost.ipynb  
  XGBoost model tested for comparison.
